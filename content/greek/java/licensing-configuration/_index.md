# Ορισμός Άδειας Java – Μαθήματα Αδειοδότησης και Διαμόρφωσης για το GroupDocs.Search Java

## Γρήγορες Απαντήσεις
- **What is the first thing to do after adding GroupDocs.Search to a project?** Φορτώστε το αρχείο άδειας ή τη ροή κατά την εκκίνηση της εφαρμογής.  
- **Which method removes watermarks and usage caps?** Ορίζοντας την άδεια με `License.setLicense(...)`.  
- **Can I retrieve a list of all file types the library supports?** Ναι, χρησιμοποιήστε το API `SupportedFormats` ή ανατρέξτε στην τεκμηρίωση.  
- **Does a licensed mode improve indexing speed?** Απόλυτα – η λειτουργία με άδεια ενεργοποιεί βελτιστοποιήσεις απόδοσης που δεν είναι διαθέσιμες στη δοκιμαστική λειτουργία.  
- **Is a separate “java licensing tutorial” needed?** Αυτός ο οδηγός και τα συνδεδεμένα μαθήματα καλύπτουν όλα όσα χρειάζεστε.

## Πώς να Ορίσετε Άδεια Java για το GroupDocs.Search

Η ρύθμιση της άδειας είναι ένα κρίσιμο μέρος οποιουδήποτε **java licensing tutorial**. Μια έγκυρη άδεια αφαιρεί τους περιορισμούς αξιολόγησης, ενεργοποιεί τη χρέωση ανά χρήση και παρέχει πρόσβαση σε premium λειτουργίες όπως **search results highlighting** και προχωρημένη ευρετηρίαση. Η διαδικασία είναι απλή: φορτώστε το αρχείο άδειας (ή τη ροή) κατά την εκκίνηση της εφαρμογής, στη συνέχεια επαληθεύστε ότι η βιβλιοθήκη αναφέρει κατάσταση άδειας πριν εκτελέσετε οποιεσδήποτε λειτουργίες αναζήτησης.

### Γιατί η σωστή αδειοδότηση είναι σημαντική

- **Compliance:** Αποτρέπει νομικά ζητήματα τηρώντας τους όρους αδειοδότησης του GroupDocs.  
- **Performance:** Η λειτουργία με άδεια ξεκλειδώνει βελτιστοποιήσεις απόδοσης που είναι απενεργοποιημένες στη δοκιμαστική λειτουργία, βοηθώντας σας να **optimize search performance** για μεγάλες συλλογές εγγράφων.  
- **Feature Access:** Ενεργοποιεί προχωρημένες δυνατότητες όπως επισήμανση αποτελεσμάτων, προσαρμοσμένη κατάταξη και ευρετηρίαση σε πραγματικό χρόνο.

### Υποστηριζόμενες μορφές αρχείων

Το GroupDocs.Search μπορεί να ευρετηριάσει και να αναζητήσει μια ευρεία γκάμα τύπων εγγράφων. Η γνώση της **list supported formats** σας βοηθά να σχεδιάσετε pipelines εισαγωγής που αποφεύγουν μη υποστηριζόμενα αρχεία και μειώνουν τα σφάλματα χρόνου εκτέλεσης. Η βιβλιοθήκη υποστηρίζει επί του παρόντος PDFs, αρχεία Microsoft Office (Word, Excel, PowerPoint), μορφές OpenDocument, απλό κείμενο, HTML και πολλά άλλα.

## Συχνές Ερωτήσεις

**Q: Χρειάζομαι άδεια για περιβάλλοντα ανάπτυξης?**  
A: Αν και μπορείτε να αξιολογήσετε τη βιβλιοθήκη χωρίς άδεια, μια άδεια ανάπτυξης αφαιρεί τις διαφημιστικές ετικέτες αξιολόγησης και σας επιτρέπει να δοκιμάσετε όλες τις premium λειτουργίες.

**Q: Πώς μπορώ να επαληθεύσω ότι η άδεια φορτώθηκε επιτυχώς?**  
A: Καλέστε `License.isLicensed()` μετά τη φόρτωση της άδειας· επιστρέφει `true` όταν η άδεια είναι έγκυρη.

**Q: Τι συμβαίνει αν προσπαθήσω να ευρετηριάσω έναν τύπο αρχείου που δεν βρίσκεται στη λίστα supported formats;**  
A: Η βιβλιοθήκη ρίχνει ένα `UnsupportedFormatException`. Χρησιμοποιήστε το tutorial supported‑formats για προ‑φίλτρου των αρχείων.

**Q: Μπορώ να αλλάξω την άδεια κατά την εκτέλεση χωρίς επανεκκίνηση της εφαρμογής;**  
A: Ναι, μπορείτε να φορτώσετε ένα νέο αρχείο άδειας χρησιμοποιώντας το ίδιο API· η αλλαγή ισχύει αμέσως για τις επόμενες λειτουργίες.

**Q: Υπάρχει τρόπος να ανακτήσω προγραμματιστικά τη λίστα των υποστηριζόμενων μορφών;**  
A: Απόλυτα—χρησιμοποιήστε τη μέθοδο `SupportedFormats.getAll()` για να λάβετε μια συλλογή όλων των μορφών που μπορεί να επεξεργαστεί η μηχανή.
