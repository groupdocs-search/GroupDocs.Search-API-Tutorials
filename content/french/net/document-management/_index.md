## Supprimer un document de l'index – Scénarios courants

Lorsqu'un fichier devient obsolète ou confidentiel, vous devrez **supprimer le document de l'index** rapidement pour maintenir la précision des résultats de recherche. Les tutoriels ci‑dessus montrent les appels API nécessaires pour supprimer des entrées sans reconstruire l'intégralité de l'index.

## Filtrer les fichiers par extension – Bonnes pratiques

Les grands dépôts contiennent souvent de nombreux types de fichiers. Utiliser les options de filtrage de fichiers de GroupDocs.Search vous permet de **filtrer les fichiers par extension** afin que seuls les formats pris en charge soient indexés, économisant de l'espace de stockage et améliorant les performances.

## Mettre en évidence les résultats de recherche – Astuces

Pour une expérience utilisateur plus riche, mettez en évidence les termes correspondants dans l'interface. Le guide « Gestion du Document Finder et mise en évidence » montre comment **mettre en évidence les résultats de recherche** directement dans les extraits retournés.

## Indexer les métadonnées de documents – Comment‑faire

Au‑delà du texte intégral, l'indexation de champs personnalisés tels que l'auteur, la date de création ou les tags améliore la pertinence. Consultez le tutoriel « Maîtriser les attributs de documents » pour des étapes pratiques afin d'**indexer les métadonnées de documents**.

## Mettre à jour les versions de documents – Flux de travail

Les documents évoluent avec le temps. Plutôt que de les ré‑ajouter, vous pouvez **mettre à jour les versions de documents** dans l'index, en préservant l'historique tout en maintenant l'index de recherche à jour. Le guide « Mettre à jour et gérer les index de documents » vous guide à travers ce processus.
