## Incremental indexing java – Что вам нужно знать

Одна из ключевых сильных сторон GroupDocs.Search — **incremental indexing java**, который позволяет добавлять или обновлять документы без полной перестройки индекса. Эта возможность снижает время простоя и повышает производительность при постоянно растущих коллекциях документов.
